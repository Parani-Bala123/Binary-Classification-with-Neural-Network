# Census Income Neural Network

Binary classification model using PyTorch to predict whether an individual earns more than $50K annually.

## Dataset
Census Income Dataset

## Model
- Embeddings for categorical features
- Batch normalization for continuous features
- Hidden layer: 50 neurons
- Dropout: 0.4

## Training
- Loss: CrossEntropyLoss
- Optimizer: Adam
- Epochs: 300
- Training samples: 25,000
- Testing samples: 5,000

## How to Run

pip install -r requirements.txt
